> A modern, efficient, and intuitive attendance management application for Android, built with the latest technologies.

AttendEZ is an Android application designed to simplify attendance tracking. Built entirely with Kotlin and Jetpack Compose, it offers a seamless and responsive user experience. It leverages modern Android development practices, including Room for local data storage and Hilt for dependency injection, to ensure the app is robust, scalable, and easy to maintain.

## Key Features

*   **Modern UI:** A clean and intuitive user interface built with Jetpack Compose.
*   **Offline First:** Uses a Room database to store data locally, ensuring functionality even without an internet connection.
*   **Efficient Data Handling:** Robust data conversion and management for entities like dates using Room Type Converters.
*   **Scalable Architecture:** Built with a clean architecture using Hilt for dependency injection, making it easy to test and scale.
*   **Easy Navigation:** Utilizes Jetpack Navigation Compose for a smooth and predictable navigation flow.

## Tech Stack & Dependencies

This project is built using a modern Android tech stack:

*   **UI:** [Jetpack Compose](https://developer.android.com/jetpack/compose) (UI, Material3, UI Tooling)
*   **Core:** [Kotlin](https://kotlinlang.org/), [Coroutines](https://kotlinlang.org/docs/coroutines-overview.html), [Flow](https://developer.android.com/kotlin/flow)
*   **Architecture:** Clean Architecture, MVVM
*   **Dependency Injection:** [Hilt](https://dagger.dev/hilt/)
*   **Navigation:** [Jetpack Navigation Compose](https://developer.android.com/jetpack/compose/navigation)
*   **Database:** [Room](https://developer.android.com/training/data-storage/room)
*   **Lifecycle Management:** [AndroidX Lifecycle](https://developer.android.com/jetpack/androidx/releases/lifecycle)
*   **Testing:** JUnit, AndroidX Test
